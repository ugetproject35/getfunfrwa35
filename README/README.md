# GetFund DeFi Project

## 概述
- 基於 Polygon 的 DeFi 投資平台
- 投資最小 10 USDT（10 的倍數）
- 每日 ROI 3%，單筆最多 150% 封頂
- 推薦鏈接 + 20 層 Matching（依 Rank 限制層數）
- Node 系統：每份 100 USDT，買入 100% 給 Admin，分紅來自投資 20% 的池，提現扣 5%，200% 出局
